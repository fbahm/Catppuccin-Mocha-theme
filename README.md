# Catppuccin-Mocha-theme
Catppuccin Mocha theme meant for Archive of Our Own and SquidgeWorld Archive.

This theme only changes the colors. Nothing too fancy. Interface and buttons remain the same. Removes the logos.

## Disclaimers:
1. I only tested this on SquidgeWorld as I use another theme on AO3, but because of its simple CSS, they should be compatible.
2. I made this on PC. Haven't tested it on mobile or big-screen platforms.
3. Made in a hurry. Could have some colors looking a bit off.
4. I have nothing to do with the official Catppuccin project. I just like the colors.

Feel free to fork and adjust to your liking. It's very half-*ssed.

Tell me if you find a bug so I can periodically update the theme to make sure its as functional as a little CSS code can be.
